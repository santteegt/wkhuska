File Structure

The file to be uploaded has to be formatted as a CSV file, with each row corresponding to a researcher and separated by commas. The first field has to be the given names of the researcher. The second field has to be the surnames and finally the third field could be void or a list of topics (or research areas) where the researcher works; separated by semi-colons (Optional).

Example: Jorge Mauricio, Espinoza Mejia, Computación; Infomática; Web Semántica; Sparql
