ACTUALIZAR REPOSITORIO DE AUTORES

Se cargara la informacion de los autores provenientes desde las Diferentes Fuentes (Endpoints) que esten activas. Las uris de los recursos (Autores) seran analizadas previamente con la sentencia ASK para verificar si los datos aun no se encuentran en la plataforma.

"Authors-Module" maneja su propio Endpoint Manager. Endpoint Manager permite agregar y eliminar los endpoints de los cuales se extraera la informacion de los autores

Los Datos de cada Endpoint, son registrados en la misma plataforma Marmotta, pero en un grafo especifico solo para endpoints

Click sobre el botton "RUN UPDATE" para iniciar la carga de autores
